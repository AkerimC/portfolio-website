# Abdulkerim Çiçek - Portfolyo

Kişisel portfolyo sitem. Terminal konseptli, koyu temalı bir React/Vite projesi. Low-level (C/C++, Assembly) ve gömülü sistemler (ESP32) başta olmak üzere geliştirdiğim projeleri sergilediğim bir vitrin.

## Özellikler
- Terminal tarzı hero tasarımı (fastfetch stili)
- Proje kartlarında tıklanabilir resim galerisi (Lightbox & Carousel)
- Koyu tema, glassmorphism detaylar ve basit animasyonlar

## Kullanılan Teknolojiler
- React 19, TypeScript, Vite
- Tailwind CSS v4
- Framer Motion

## Geliştirme
Projeyi lokalde çalıştırmak için:

```bash
git clone https://github.com/AkerimC/portfolio.git
cd portfolio
npm install
npm run dev
```

Build almak için:
```bash
npm run build
```
